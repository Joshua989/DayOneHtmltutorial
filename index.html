CSS Fundamentals

What is CSS?

CSS (Cascading Style Sheets) is a styling language used to describe the presentation of HTML documents. It controls the layout, colors, fonts, and overall appearance of web pages.

CSS allows you to separate the content (HTML) from its presentation, making your code more maintainable and flexible.

CSS Types

1. Inline CSS

Inline CSS is applied directly to HTML elements using the style attribute.

This paragraph uses inline CSS styling!

2. Internal CSS

Internal CSS is defined within a <style> tag in the HTML document's <head> section.

This entire page is mostly styled using internal CSS!

3. External CSS

External CSS is stored in separate .css files and linked to HTML documents.

Example: <link rel="stylesheet" href="styles.css">

CSS Selectors

  • Element Selector: Selects all instances of a specific HTML element (e.g., p { color: blue; })
  • Class Selector: Selects elements with a specific class attribute (e.g., .section { background: white; })
  • ID Selector: Selects a single element with a specific ID (e.g., #css-types { border-left: 5px solid red; })
  • Grouping Selector: Applies the same styles to multiple selectors (e.g., h3, h4 { color: #2c3e50; })
