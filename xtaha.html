What is Semantic HTML?

Semantic HTML uses HTML elements that clearly describe their meaning to both the browser and the developer. Using semantic markup helps with:

  • Accessibility
  • SEO (Search Engine Optimization)
  • Readability of code
  • Code maintenance

Sectioning Elements

These elements define sections in an HTML document:

Article Element

The <article> element represents a self-contained composition in a document, page, application, or site, which is intended to be independently distributable or reusable.

Section Element

The <section> element represents a standalone section of a document, which doesn't have a more specific semantic element to represent it.
